# 本造商行 BENZAO ｜ 鎖緊產業斷層的數位指揮官

![BENZAO Banner](https://raw.githubusercontent.com/iceoo9621215/benzao-merchant/main/assets/pvd-screws.png)

## 專案簡介
本造商行（BENZAO）是台灣精密扣件的「微型客製」賦能者。我們致力於將傳統機械產業的斷層數位化，透過 **C2M (Consumer to Manufacturer)** 模式，將客戶的創意與草圖轉化為 ±0.05mm 航太級的精品零件。

### 核心價值
- **微型客製**：打破傳統工廠接單門檻，實現少量多樣的高品質生產。
- **數位指揮**：將繁瑣的手繪草圖數位化，優化生產鏈。
- **精品規格**：提供具有品牌質感的 PVD 鍍層與精品零件。

## 網站功能與特點
- **動態展示**：展示產品細節與 0.05mm 的精密工藝。
- **C2M 流程模擬**：視覺化呈現從設計到生產的自動化流程。
- **SOS-V 募資計畫**：展示目前的產品方案與市場動向。
- **沉浸式 UI**：採用工業精品風格（Industrial Premium）設計，搭配 Orbitron 字體與金屬質感色調。

## 技術架構
- **前端核心**: HTML5, CSS3 (Vanilla CSS with Design System)
- **邏輯控制**: JavaScript (Vanilla JS)
- **視覺元素**: 
    - [Lucide Icons](https://lucide.dev/) - 向量圖示
    - [Google Fonts](https://fonts.google.com/) - Noto Sans TC & Orbitron
- **設計風格**: Industrial Premium / Dark Mode / High-Tech Glassmorphism
